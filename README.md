# starlight-os
nixos based linux operating system for software development

WARNING!!! The partition script in this repository could ERASE ALL OF YOUR DATA!!!

principles:
  be vi-like
  be minimal

 keybindings:
 	super + arrow:
 		change focus
 	super + number:
 		change desktop
 	super + shift + arrow:
 		move window
 	super + shift + number:
 		move window to desktop

 	super + space:
 		menu
 	super + enter:
 		terminal

 	super + q:
 		quit
 	super + shift + q:
 		logout...
 	super + control + q:
 		shutdown...

 	super + h:
 		flip 180
 	super + shift + backspace:
 		reload bar/keybindings
 	super + backslash:
 		ranger
